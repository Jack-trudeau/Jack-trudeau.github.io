Written Response

One problem we had when creating this program was the while loop wouldn’t break even though we were breaking it. It would take choosing exit twice to actually exit the loop. We never figured out why it wasn’t working but we figured out that it worked perfectly when we set the while loops variable name outside of the function. We think it might have been running the function again and setting the while loop to true again. Another difficulty we encountered was when writing the code we already had the key generator, the encryptor and the decryptor done we had to link them all together. We solved this by creating a menu function named welcome() this was just a simple input function where you choose what you want to do by typing the letter that corresponds to it. A part of the program i completed completely independently was the encryptor function.
