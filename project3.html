When developing this program something i completed independently was writing the code for the specific accounts. One problem i had when developing this code was figuruing out how to do a transfer function, i ended up making the variables global so that they can be changed outside of functions, i also made each account their own function, this fixed my problem of not knowing how to transfer money between accounts. A second problem i had when writing this program was we wanted to make it possible to make a new account, log out go to other accounts and come back to the account that you made. We did this by making two seperate functions for initially making the new account, and returing to the new account. I made a check where i set a variable to true when the account was initially made so when you went back to the account the program knew that the account was already made and it goes to the second function.
